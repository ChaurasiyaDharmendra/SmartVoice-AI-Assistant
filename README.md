# 🤖 SmartVoice-AI-Assistant

A powerful AI-based Virtual Assistant built using the MERN stack (MongoDB, Express, React, Node.js).  
This assistant can **listen, understand, and respond like a real AI assistant** with voice interaction and smart commands.

---

## 🚀 Features

- 🎤 Voice Input (Speech Recognition - Web Speech API)
- 🔊 Voice Output (Text-to-Speech)
- 🧠 Smart AI Responses (OpenRouter GPT)
- 🌐 Hindi + English Support (Auto Language Detection)
- ⚡ Smart Command Execution (YouTube, Google, Instagram, etc.)
- 🔐 User Authentication (JWT + bcrypt)
- 🧑‍💻 Personalized Assistant (Name + Image)
- 🖼️ Image Upload (Cloudinary + Multer)
- 📱 Fully Responsive UI
- ☁️ Deployable on Render
