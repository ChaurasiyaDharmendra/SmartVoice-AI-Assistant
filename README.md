# SmartVoice-AI 🤖🔥

Build your own AI Virtual Assistant using the MERN Stack (MongoDB, Express, React, Node.js). This project is inspired by a JARVIS-style assistant that can talk, think, and interact like a real AI.

---

## 🚀 Features

- 🎤 Voice Input (Speech Recognition - Web Speech API)
- 🔊 Voice Output (Text-to-Speech)
- 🤖 Smart AI Responses (Gemini AI)
- 🔐 User Authentication (JWT + bcrypt)
- 🧠 Personalized Assistant (Name + Image)
- 📸 Image Upload (Cloudinary + Multer)
- 📱 Fully Responsive UI
- 🌐 Deployable on Render
